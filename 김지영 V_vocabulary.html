📖 학생용 시스템 사용 매뉴얼
🔐 1. 시스템 로그인 방법
최초 접속 시: 좌석번호, 이름, 그리고 선생님이 알려주신 비밀번호를 정확히 입력합니다.
이후 접속 시: 시스템이 이름을 기억합니다. 비밀번호만 입력하고 학습을 바로 시작하세요. (다른 사람이 쓰던 자리라면 '다른 학생으로 로그인하기'를 눌러주세요)
📝 2. 이든 단어 TEST 진행 순서
1차 테스트: 정답을 입력하고 채점하세요. 틀린 단어는 자동으로 다음 2차 테스트 대기열로 넘어갑니다.
2차/3차 테스트: [테스트 시작] 버튼을 눌러 이전 단계의 정답을 가리고, 틀렸던 단어만 모아서 다시 시험을 봅니다.
오답 처리: 뜻이 여러 개인 경우 일부만 적으면 정답 처리되지만 파란색으로 표시됩니다. 완전히 틀린 경우 빨간색으로 오답노트에 기록됩니다.
⌨️ 3. 탭별 학습 요령
[1. 뜻 쓰기 테스트] 단어의 뜻을 한글로 적습니다. 뜻이 여러 개라면 쉼표(,)로 구분해서 적으세요. (띄어쓰기는 무시됩니다.)
[2. 철자 쓰기 테스트] 한글 뜻을 보고 영어 단어 스펠링을 정확하게 입력하세요.
[3. 철자 맞추기 게임] 박스에 있는 알파벳을 올바른 순서대로 클릭하여 단어를 완성하세요. 완료 시 원어민 발음과 플래시카드가 나타납니다.
🚀 4. 학습 결과 전송 (가장 중요!)
모든 학습과 테스트가 끝나면 화면 맨 아래에 있는 [오늘 학습 결과 구글폼으로 전송하기] 보라색 버튼을 반드시 눌러야 합니다.
전송하지 않고 창을 닫으면 다음 접속 시 경고창이 나타나며, 이전 기록을 전송해야 새 학습을 시작할 수 있습니다.
